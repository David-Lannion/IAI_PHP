# IAI_PHP

## Mise en place
Pour générer l'image PHP
```shell
docker-compose build
```
Puis pour lancer les différentes images :
```shell
docker-compose up -d
```

Pour relancer :
```shell
docker-compose down && docker-compose up -d
```
 ## Developpement
 Les fichiers PHP seront à mettre dans le dossier ./php
